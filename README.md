# JavaLab6 

## Опис
Реалізація типізованої колекції в Java для лабораторної роботи. Колекція підтримує:
- Масив із початковою кількістю елементів 15.
- Збільшення розміру масиву на 30% за потреби.
- Повну реалізацію інтерфейсу `List`.
